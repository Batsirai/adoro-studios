# Adoro Studios — AI Marketing & Sales Agency

> **Autonomous creative agency powered by AI agents. No contractors. No bottlenecks.**

Adoro Studios is a fully AI-operated marketing and sales agency. We generate, schedule, and optimize content at scale using a multi-agent system coordinated by Zo Computer.

## What We Do

- **UGC Content Pipelines** — AI-generated short-form video scripts and branded images for TikTok, Instagram, X
- **Social Media Marketing** — Multi-platform posting and scheduling via Buffer API
- **Copywriting** — Ads, emails, landing pages, social captions
- **Brand Voice** — Consistent emotional architecture across all content
- **Sales Automation** — Lead capture, nurture, and conversion optimization
- **Analytics** — PostHog-powered performance tracking with daily iteration

## AI Agents

| Role | Function |
