# Marketing Program Skill

Marketing strategy and execution for Adoro Studios clients.

## Core Capabilities

1. **Social Media Marketing**
   - Multi-platform content calendar management
   - Post scheduling via Buffer API
   - Engagement monitoring and response strategy
   - Hashtag research and optimization

2. **UGC Content Pipeline**
   - Script generation using emotional hook frameworks
   - AI image generation (Nano Banana 2, Imagen 4)
   - Branded overlay composition (logo, fonts)
   - Video asset preparation (Veo 3.1 for motion)

3. **Copywriting**
   - Ad copy (Facebook, Instagram, TikTok, Google)
   - Email sequences (welcome, nurture, abandoned cart)
   - Landing page copy
   - Social captions with CTAs

4. **Brand Voice**
   - Maintain consistent tone across all channels
   - AlreadyLoved: warm, observational, emotionally safe
   - No preaching, no guilt, no urgency tactics

5. **Analytics & Optimization**
   - PostHog event tracking for content performance
   - Weekly metrics review
   - Hook A/B testing (hero headlines, ad copy variants)
   - Conversion funnel analysis

## Hook Patterns (AlreadyLoved)

### Approved Hook Families
- **Confessional** (child POV): "Can I tell you something I don't say out loud?"
- **Observational** (toy/parent POV): "She holds me tighter when..."
- **Interpretive** (meaning-making): "When you say ____, I learn ____."
- **Secure Identity**: "I'm still okay."
- **Unspoken Question**: "Do I need to be easier to love today?"

### Story Situations
- Morning rush, bedtime fear, public correction
- Time-out, sibling comparison, being told "not now"
- Phone distraction, messes, emotional overwhelm
